Wordy Readme:

Implementations:
1. I created one new interface and new implementation class for doing DFS search of character board.(extra credit)
2. When click "Score Me", the strings will be verified by both methods "isRealWord" & "isWordInBoard".
3. The character board is generated by using Java Random class's nextInt that provided equal probability for each character.

Run:
1. build.xml is provided with the solution.
2. junit lib is provided with the solution.

To run unit test only:
-> type "ant junit" under the root directory.
ex: $ ant junit

To run unit test & program:
-> type "./Wordy"
ex: $ ./Wordy
